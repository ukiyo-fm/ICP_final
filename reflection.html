Short Conclusion

This project showed me that real leadership development can happen through a very practical task such as learning web development. By working with HTML, CSS and JavaScript I trained my patience, adaptability and emotional control. I also learned that I can handle unfamiliar, technical content if I break it down and stay consistent.

I now see coding not only as a digital skill but as a way to practise problem-solving and self-management, which are essential for a future manager.
